# Simulation Argument and Theological Implications Source Cluster

## Purpose

This folder holds raw source files for the Simulation Argument evidence cluster before claim extraction or workbook updates.

## Cluster Working Title

CLUST-SIM-001 - Simulation Argument and Theological Implications

## Core Question

If simulated worlds are possible or likely, what does that imply for theism, naturalism, creation, divine hiddenness, incarnation, moral responsibility, religious experience, consciousness, and theological anthropology?

## Folder Guide

### discord_thread/

- Discord exports or cleaned text from the Christians vs Atheists thread between Eric / EvilWizard and DuncanIdaho.
- Preserve speaker attribution.
- Do not treat every claim in the thread as Eric's view.

### articles/

- Scholarly papers, academic articles, and scholar-written blog responses.
- Include Bostrom's original argument and major critiques/responses where available.

### youtube/

- YouTube transcripts or cleaned transcript text related to the Simulation Argument.
- Include clips or discussions by Sam Harris, S. James Gates Jr., and similar sources.
- These are usually context/discovery unless especially substantive.

### notes/

- Eric's own notes, summaries, questions, or reading reflections.
- Keep these distinct from external source documents.

## Workflow Note

Files placed here should later be bulk registered, assigned source IDs, and added to CLUST-SIM-001 using the evidence cluster tools. Do not push every file directly into claim extraction. Triage and cluster mapping come first.

## Privacy Note

These are real research/source files and should remain local/ignored unless deliberately converted into sample data.
