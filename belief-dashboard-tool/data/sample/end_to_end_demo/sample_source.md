# Sample Source: Explanation Scope And Theory Choice

This neutral demo source contains placeholder philosophical material for exercising the CLI workflow. It is not private, personal, medical, religiously personal, or based on a real user's beliefs.

The sample argument says that explanation scope can matter when comparing theories, because a theory that accounts for more observations with fewer special exceptions may deserve additional attention.

The sample objection says that anecdotal evidence can be over-weighted when it feels vivid, so a careful review process should distinguish salience from reliability.

The sample reflection says that hypotheses can be compared without assuming certainty. A reviewer can assign provisional weights, record uncertainty, and revisit the update later.
